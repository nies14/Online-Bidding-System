# Web_OnlineBiddingSystem

The online bidding system is a bidding service provider in online.  It was made by usingHTML, CSS, PHP, JavaScript, MySQL etc.  Three departments are there.  Admin, Employee  Customer.  WhereAdmin can add, delete, update both employee  product.  An employee can add, delete, update a product.  And thecustomer can sign in, sign up and place a bid for a product.  After a certain time, whose bidding amount is highest will be declared as the winner.  Customers cannot bid less than the starting bidding amount.
